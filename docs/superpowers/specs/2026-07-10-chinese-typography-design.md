# ADING 摄影作品集 — 中文排版变更规格

## 目标

将现有英文优先网站改为中文优先，同时保留 Modern Atelier 的冷银灰、非对称摄影编排和硬朗视觉识别。变更重点是中文字体、字号、字距和行高，不改变页面架构、作品数据结构、GitHub Pages 地址或交互范围。

## 语言

- HTML 语言设为 `zh-CN`。
- 品牌署名继续使用 `ADING / PHOTOGRAPHY`。
- 导航、页面标题、系列名称、系列说明、About、404、灯箱控制、无障碍名称和元数据全部使用简体中文。
- 年份、图片序号和品牌英文保持拉丁字符。
- 不采用中英并排，避免摄影页面信息密度过高。

## 字体体系

- **品牌英文与数字造型：** Antonio Variable。
- **所有中文标题与正文：** Noto Sans SC Variable（思源黑体体系），本地打包，不依赖第三方字体请求。
- **纯数字和短英文元数据：** IBM Plex Mono。
- **系统回退：** `PingFang SC`、`Hiragino Sans GB`、`Microsoft YaHei`、无衬线系统字体。

中文正文不使用负字距。中文展示标题使用 `0.01em` 到 `0.04em` 的轻微正字距，避免粗体笔画拥挤。仅英文品牌署名保留宽字距。

## 字号与行高

| 角色 | 桌面 | 手机 | 行高 |
|---|---:|---:|---:|
| 首页主标题 | 96–176px 自适应 | 58–82px 自适应 | 0.95–1.0 |
| 系列大标题 | 72–132px 自适应 | 48–68px 自适应 | 1.0 |
| About 主标题 | 64–128px 自适应 | 44–64px 自适应 | 1.08 |
| 正文 | 16–18px | 16px | 1.75 |
| 导航 | 13px | 13px | 1.4 |
| 元数据与说明 | 12px，禁止更小 | 12px | 1.6 |

正文最大阅读宽度收窄到约 32–36 个中文字符。字号随视口变化，但不随内容任意缩小；手机端宁可换行，也不压缩到 16px 以下。

## 首页文案和构图

- 主标题改为“凝视 / 之间”，分两行并仅在首页穿过主照片边界。
- “精选作品”“当前系列”“查看作品”等结构标签中文化。
- 系列示例改为“静默时刻”“雨后”“星期日的窗边”。
- 标题覆盖照片仍是唯一显著设计动作；其他页面不增加装饰。

## 交互和无障碍

- 中文化 skip link、导航 aria-label、图片打开按钮、灯箱名称、上一张、下一张、关闭和计数播报。
- 所有触控目标继续保持至少 44px 高。
- 保持 Escape 关闭、方向键换图和 reduced-motion 支持。
- 保持中文内容下唯一 `h1`、顺序标题层级和可见焦点。

## 性能

- 只打包需要的 Noto Sans SC 可变字重样式。
- 保留系统字体回退与 `font-display: swap`，避免字体加载时白屏。
- 不添加新的脚本、动画库或第三方字体服务。

## 验证

- Astro 类型检查与生产构建零错误。
- 375、768、1024、1440px 下验证标题不裁切、正文不小于 16px、元数据不小于 12px且无横向溢出。
- 验证中文字体实际加载，About、404、系列页和灯箱均无残留英文界面文本。
- 重新部署 GitHub Pages，并验证线上首页返回 HTTP 200、字体资源加载成功、浏览器控制台无错误。
